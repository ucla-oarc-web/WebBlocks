Base/Block

WebBlocks provides float-left and float-right classes to floating a block-level element left or right. It also provides a float-clear class that will clear all floats.

.float-left
.float-right
.float-clear

Floats within Container

WebBlocks provides two different classes that can be used to contain floated elements. The float-container uses overflow:auto, which will work in most cases, but in cases where child elements exceed the area of the containing element, then the clearfix class must be used.

.float-container

.float-left

.clearfix
