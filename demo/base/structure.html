Container

WebBlocks provides a fluid container class (which is in use here). It can be applied to the body to contain the entire page contents or to a smaller subset of elements.

Right-oriented Variant of Row

The row collapse at small breakpoint usually occurs with the left-most element first in the set. However, the class set .row.right may be used to orient the panels from right-to-left rather than left-to-right, allowing collapse preference to the right-most element first.

Grid System Collapse Controls

The default behavior for a row is that it collapses at $breakpoint-small. However, this is not always the intended behavior.

No Collapse

If the grid should never collapse, use .row.no-collapse:

.row.no-collapse .panel-6
.row.no-collapse .panel-6

Collapse for xxsmall

If the grid should collapse at $breakpoint-xxsmall, use .row.xxsmall-collapse:

.row.xxsmall-collapse .panel-6
.row.xxsmall-collapse .panel-6

Collapse for xsmall

If the grid should collapse at $breakpoint-xsmall, use .row.xsmall-collapse:

.row.xsmall-collapse .panel-6
.row.xsmall-collapse .panel-6

Collapse for small

If the grid should collapse at $breakpoint-small, the use of .row.small-collapse is optional (this is the default collapse behavior):

.row.small-collapse .panel-6
.row.small-collapse .panel-6

Collapse for medium-small

If the grid should collapse at $breakpoint-medium-small, use .row.medium-small-collapse:

.row.medium-small-collapse .panel-6
.row.medium-small-collapse .panel-6

Collapse for medium

If the grid should collapse at $breakpoint-medium, use .row.medium-collapse:

.row.medium-collapse .panel-6
.row.medium-collapse .panel-6

Collapse for medium-large

If the grid should collapse at $breakpoint-medium-large, use .row.medium-large-collapse:

.row.medium-large-collapse .panel-6
.row.medium-large-collapse .panel-6

Large Collapse

If the grid should collapse at $breakpoint-large, use .row.large-collapse:

.row.large-collapse .panel-6
