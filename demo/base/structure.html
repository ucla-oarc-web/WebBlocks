Base/Structure

Container

WebBlocks provides a fluid container class (which is in use here). It can be applied to the body to contain the entire page contents or to a smaller subset of elements.

Grid System

WebBlocks provides a grid system through the use of the row and panel-X classes. By default (and thus in this demo), a row is twelve panels wide.
