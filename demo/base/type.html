Base/Type

WebBlocks provide a default typography that is a combination of a Normalize.css (a reset stylesheet) and a set of font definitions based on proportional em-based font sizes and pixel-based line spacing. This mixture both scales responsively and maintains a consistent line spacing.

UI framework adapters override these definitions with their own.
