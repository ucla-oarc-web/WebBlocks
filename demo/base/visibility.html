Base/Visibility

While responsive design should not often code explicitly for different screen sizes, in some cases, this is necessary. As such, WebBlocks provides a set of classes prefixed with hide- and show- for the sizes small, medium, large and xlarge.

.hide-small
.show-small
.hide-medium
.show-medium
.hide-large
.show-large
.hide-xlarge
.show-xlarge
.hide-media-query
