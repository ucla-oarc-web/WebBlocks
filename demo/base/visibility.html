Base/Visibility

While responsive design should not often code explicitly for different screen sizes, in some cases, this is necessary. As such, WebBlocks provides a set of classes prefixed with hide- and hide-above- to accommodate this. Essentially, the .hide-small class hides content at and below the $breakpoint-small size, and .hide-above-small hides content above the $breakpoint-small.

.hide-xxsmall .hide-above-xxsmall
.hide-xsmall .hide-above-xsmall
.hide-small .hide-above-small
.hide-medium-small .hide-above-medium-small
.hide-medium .hide-above-medium
.hide-medium-large .hide-above-medium-large
.hide-large .hide-above-large
