Base/Visibility

While responsive design should not often code explicitly for different screen sizes, in some cases, this is necessary. As such, WebBlocks provides a set of classes prefixed with hide- to accommodate this. Essentially, the .hide-small class hides content at and below the $breakpoint-small size.
