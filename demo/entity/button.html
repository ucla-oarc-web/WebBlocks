Variations

Separate Moods on Text

The a.button and button.button elements both support changing text of color by using an internal span.text.{mood} element. This is not possible with an input element.

a.button .text.danger button.inverse .text.highlight

Size Controls

The button supports mini, small and large size modifiers.

a.button.mini

a.button.small

a.button.large
