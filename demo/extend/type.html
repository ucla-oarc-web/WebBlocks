Extend/Base/Type

This is an extended library for Base/Type.

Extend/Base/Type/Split

The Extend/Base/Type/Split library makes it easy to apply typographical effects such as kerning, tracking and leading via the "lettering" package (Lettering.js). It creates a span around each character if using .text-split-chars, word if using .text-split-words and line if using .text-split-lines.

For example, if character splitting is used around "abc", then "b" can be styled with .char2 { .. } or, if "ab cd ef" is used with word splitting, then "ef" may be styled with .word3 { .. }.

This library requires the "lettering" package:

WebBlocks.config[:build][:packages] << :lettering

Variations

When using .text-split-chars.text-split-words, each word is assigned a class .wordX and each character is assigned a class .charY

When using .text-split-chars.text-split-lines,
each line is assigned a class .lineX
and each character is assigned a class .charY

When using .text-split-words.text-split-lines,
each line is assigned a class .lineX
and each word is assigned a class .wordY

When using .text-split-chars.text-split-words.text-split-lines,
each line is assigned a class .lineX,
each word is assigned a .wordY,
and each character is assigned a class .charZ
