Masthead Tests

  • The Masthead block (.-masthead) is the area shown in black. White nav areas are structurally not part of the Masthead.
  • Nesting hierarchy and names changed to reflect WebBlocks markup requirements
  • WebBlocks names have a prefix dash: '-' ; Original Bootstrap Classes reflect dependencies - these names are not part of the masthead scope so they have been left in original classnames for the sake of clarity -they also show where classes need to be abstracted apart
  • .-nav.-nobar is introduced to cover the use-case when a horizontal nav is a child of the .-masthead; this distinct class name is only used for the static CSS to demo the need of a @mixin that targets .-nav.-bar when parent is .-masthead. Extra styles have been added to the ModBootstrapNav.css where needed.
  • .-sitetitle is introduced (was .brand in Bootstrap) to demo @mixin and show changes needed when .-sitetitle is a child of .-masthead vs. .-nav.-main
  • Only basic implementation cases are show here; for full Masthead patterns see Patterns mockups and for the inherited nav details see the navbarCSS docs
